Для инициализации репозитория пишем git init

Для понятия текущего состояния репозитория пишем git status

Чтобы добавить файл в staging area пишем git add <имя файла>

Чтобы сделать commit пишем git commit <имя файла>

Чтобы посмотреть историю коммитов пишем git log

Чтобы перейти к конкретному коммиту пишем git checkout ХЕШ
